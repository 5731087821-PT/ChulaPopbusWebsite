WHAT IS CU POP BUS APP?
Never get lost in Chula again
Chula Pop Bus App helps making navigation through Chulalongkorn University much
easier.
Hey, where's the bus?
WHAT ARE THE MAIN FEATURES?
My Location
The application can display your current location and also the nearest station to your current location!

Bus Routes
By selecting the routes number, you can choose to display different routes on the map.
Bus Tracking
The application can track real-time position of the buses with its speed (km/hr), status, and current route.

Search
The name of the stations can be searched with tags which shows the routes which passes through those stations.
Sir, how do I get to MBK?
WHY DO WE NEED IT?
Chula Pop Bus application helps saving time. By using the Chula Pop Bus Application, instead of pointlessly hurrying to catch a Pop Bus, users can use the application to estimate how long it will take for the next Pop Bus to reach the station. Outsiders who come to visit Chula can also use the app so they won’t get lost or confused about the different bus routes. Chula Pop Bus Application helps people in Chula to save time and make their lives more convenient.

DOES IT REALLY HELP?
This application does help a lot. According to statistic recordings, the number of download, active users and feedback in both Android and iOS platform, we can see that the majority of Chula citizen use this application everyday.
